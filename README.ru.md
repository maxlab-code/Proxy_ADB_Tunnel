# 📖 Руководство пользователя: Утилита Proxy ADB Tunnel

> [!IMPORTANT]
> 🌐 Read this in other languages:
> [English](README.md) | [Русский](README.ru.md)

## 1. Назначение утилиты
**Proxy ADB Tunnel** — это портативная связка скриптов для Windows, предназначенная для бесшовного проброса VPN-трафика со смартфона Android на персональный компьютер через USB-кабель.

**Ключевые преимущества:**
* **Изоляция от провайдера:** В отличие от стандартного режима «USB-модем», трафик идет не через сетевые интерфейсы, а инкапсулируется в отладочный мост (ADB). Это полностью исключает утечки DNS (DNS leaks) и блокировки со стороны оператора связи.
* **Полная автоматизация:** Скрипт самостоятельно запрашивает права Администратора, запускает ядро NekoBox/NekoRay в фоновом режиме (в системном трее) и автоматически восстанавливает соединение при физическом переподключении кабеля телефона.
* **Портативность:** Утилита использует относительные пути. Всю папку программы можно свободно перемещать между дисками или использовать на флешке без перенастройки.

## 2. Структура каталогов
Для корректной работы утилиты папки должны располагаться в следующем порядке:

```text
📁 Любая_Ваша_Папка/
 ├── 📁 nekoray/                 # Папка клиента NekoBox/NekoRay
 │    ├── nekobox.exe            # (или nekoray.exe)
 │    └── ... (файлы программы)
 │
 └── 📁 Proxy_ADB_Tunnel/        # Папка с нашей утилитой
      ├── Tray_ADB_Proxy.ps1         # Главный скрипт запуска и управления (в трее)
      └── Auto_Proxy.bat         # Фоновый процесс проброса портов ADB
```

## 3. Первоначальная настройка процесса

### Шаг А: Подготовка Android-смартфона
1. Включите **Отладку по USB** (USB Debugging) в параметрах разработчика на телефоне.
2. Откройте ваше VPN-приложение (Proxy, v2rayNG или аналог) и перейдите в его настройки.
3. Найдите параметр **SOCKS5 Port** и убедитесь, что он установлен на `10808` (или измените порт в файле `Auto_Proxy.bat`, если приложение использует другой).
4. Запустите VPN-соединение в приложении.

### Шаг Б: Подготовка NekoBox на ПК
*Эту настройку нужно выполнить всего один раз.*
1. Запустите NekoBox вручную.
2. В верхнем меню выберите **Профиль** -> **Добавить профиль** -> **SOCKS**.
3. Заполните данные профиля:
   * **Имя:** ADB Tunnel (или любое другое)
   * **Адрес (Address):** `127.0.0.1`
   * **Порт (Port):** `10808` *(этот локальный порт резервирует наш скрипт)*
4. Нажмите **ОК**.
5. Кликните правой кнопкой мыши по созданному профилю и нажмите **Запустить** (Start).
6. Включите галочку **Режим TUN** (TUN Mode) в интерфейсе программы.
7. Полностью закройте NekoBox (выйдите из него в системном трее).

## 4. Повседневное использование

1. Подключите смартфон к ПК с помощью USB-кабеля.
2. При первом подключении к ПК телефон может запросить разрешение на отладку — нажмите «Разрешить всегда для этого компьютера».
3. Включите режим модема на смартфоне.
4. Запустите файл `Tray_ADB_Proxy.ps1`.
   * *Примечание:* Если система запросит права Администратора (Контроль учетных записей / UAC) — нажмите **«Да»**. Это необходимо для работы режима TUN в NekoBox.
5. В системном трее (возле часов) появится иконка командной строки.
6. Готово! Скрипт автоматически запустит NekoBox в свернутом виде и поднимет туннель. Весь трафик Windows теперь маршрутизируется через телефон.

---

## 5. Управление утилитой (Меню в трее)
Клик правой кнопкой мыши по значку в трее открывает меню управления:

* **▶ Запустить авто-туннель:** Принудительно запускает фоновый процесс ожидания кабеля и стартует NekoBox, если они были закрыты.
* **👁 Показать / Скрыть консоль (или Двойной клик по иконке):** Позволяет увидеть окно с логами работы ADB. Полезно для диагностики: показывает, ждет ли скрипт телефон или порт уже успешно проброшен. Повторное нажатие прячет окно обратно в фон.
* **⏹ Остановить:** Полностью отключает проброс портов, завершает фоновые процессы ADB и закрывает NekoBox. Скрипт при этом остается в трее.
* **❌ Выход:** Останавливает все процессы туннеля, закрывает NekoBox и полностью выгружает саму утилиту из памяти ПК.
